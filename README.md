# Eckl Eco Technics - Materialverwaltung V0.9.7

Basis: v0.9.6 im Master-Design, weitergeführt als v0.9.7.

## Neu in v0.9.7

- Automatische Groß-/Kleinschreibung bei Material anlegen.
- Automatische Groß-/Kleinschreibung bei Admin-Korrektur.
- Materialbeispiele werden sauber normiert, z. B. `almg3` → `AlMg3`, `v2a` → `V2A`, `dc01` → `DC01`, `s235` → `S235`.
- Teilenr. wird automatisch großgeschrieben, z. B. `t-12345` → `T-12345`.
- Normierung läuft im Browser und zusätzlich serverseitig.
- Werkstoffnummern werden automatisch mit Punkt gespeichert, z. B. `1,4404` → `1.4404`, `1,4571` → `1.4571`, `1,4301` → `1.4301`.
- Vorhandene Funktionen aus v0.9.5 bleiben erhalten: Sonderformat und Teilenr.

## Start

1. ZIP entpacken.
2. `3_BROWSER_SERVER_TESTEN.bat` starten.
3. Im Browser anmelden und Material anlegen oder als Admin korrigieren.

Master-Design bleibt unverändert.
