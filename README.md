# Eckl Eco Technics – Materialverwaltung

### v3.3 Laser-Bestellung & Resttafeln

- Laser-Bestellung ist schlanker: kein Lieferant und kein Werkszeugnis im Anfragefenster.
- Büro/Chef/Admin wählen den Lieferanten beim Klick auf **Bestellt** aus einer Liste oder geben ihn frei ein.
- Resttafeln können mit Material, Stärke, Format/Sonderformat, Menge und Lagerplatz angelegt werden.
- Master-Design bleibt unverändert.


V3.3: Bestellungen werden nach Lieferdatum und Lieferant sortiert. Beim Bestellt-Melden wählt Büro/Chef/Admin das Lieferdatum. Das Bestelldatum bleibt sichtbar. Leere Wareneingangspositionen werden nach direkter Entnahme automatisch entfernt.
