# Teständerungen V2.3

- In Bestellungen nach einem Lieferanten suchen: Treffer werden unter dem Lieferanten zusammengefasst.
- Prüfen, dass in dieser Ansicht das Datum direkt vor dem Material steht.
- Lieferung buchen und anschließend mit „Geliefert rückgängig“ zurücknehmen.
- Prüfen, dass die Bestellung weiterhin sichtbar bleibt und wieder den Status „Bestellt“ hat.
- Als Büro/Chef/Admin ein Material über „Komplett ausbuchen“ auf 0 setzen.
- Als Laser prüfen, dass „Komplett ausbuchen“ nicht sichtbar ist.
